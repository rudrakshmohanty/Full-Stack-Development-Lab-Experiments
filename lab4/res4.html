The Power of clamp()

Using just viewport width (vw) can make text unreadably small on phones or gigantic on large monitors. The clamp() function solves this by setting boundaries.

The syntax is clamp(MIN, PREFERRED, MAX):

  • MIN: The absolute minimum font size.
  • PREFERRED: The ideal size that grows with the screen (e.g., 1rem + 0.5vw).
  • MAX: The absolute maximum font size.
